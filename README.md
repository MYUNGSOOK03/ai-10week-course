# AI MEKKUM LIFE+ 10주 코스

🌿 검색·글쓰기·콘텐츠·자동화를 통합한 실전형 AI 루틴 학습 플랫폼

## 시작하기

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

### 3. 빌드
```bash
npm run build
```

### 4. 프리뷰
```bash
npm run preview
```

## 기술 스택
- React 18
- Vite
- Tailwind CSS
- Framer Motion
